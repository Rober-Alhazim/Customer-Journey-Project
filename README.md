# Customer Journey Recommendation System

## Goal
Recommend the **top 4 sales actions** (Email, Meeting, Demo, etc.) to maximize the chance of **winning a sales opportunity**, personalized by:
- **Country** (e.g., US, UK, FR)
- **Solution** (e.g., MRS, Digital)

## Files
- `data_cleaning.py` → Cleans raw CRM data
- `path_analysis.py` → Finds top actions per segment
- `decision_tree_model.py` → Trains interpretable ML model
- `recommendation_system.py` → Full recommendation engine
